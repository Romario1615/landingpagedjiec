Drop your own carousel photos here, named exactly:

  1.jpg
  2.jpg
  3.jpg
  4.jpg
  5.jpg

That's it — no code changes needed. Each hero/catalog carousel slide looks
for one of these files automatically. Any slide whose file is missing just
shows a dashed "Espacio para imagen" placeholder instead of a broken image,
so it's safe to add them one at a time.

Only .jpg is supported right now (see CAROUSEL_IMAGE_FOLDER in
assets/site.js if you need a different extension or more than 5 slots).
